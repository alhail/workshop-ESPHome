Willkommen zum GitHub Workshop

Workshop Details

Dieser Workshop bietet eine Einführung in Git und GitHub. Sie lernen, wie Sie Repositorys erstellen, Änderungen verfolgen und effektiv mit anderen Entwicklern zusammenarbeiten.

Ablauf

  • 09:00 - 10:00: Einführung in Git
  • 10:00 - 11:00: Grundlegende Befehle und Konzepte
  • 11:00 - 12:00: Arbeiten mit GitHub
  • 12:00 - 13:00: Mittagspause
  • 13:00 - 14:00: Fortgeschrittene Techniken
  • 14:00 - 15:00: Zusammenarbeit und Pull Requests
  • 15:00 - 16:00: Q&A und Abschluss
